About Us

What is Gotchu MRS ?

Gotchu MRS is a music recommender system developed for a final year project using Spotify's Web API. It aims to generate more accurate recommendations with the help of AI and machine learning.


What makes it different ?

There are a lot of similar applications today, but what makes Gotchu different is that other applications directly takes the recommendations from Spotify while we use machine learning algorithms to make recommendations for you.
